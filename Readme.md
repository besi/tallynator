tallynator
==========

Create tally images in a jiffy.

Setup
-----

	cd /path/to/project
	brew install imagemagick
	gem install bundler
	bundle install

See [Homebrew](http://mxcl.github.com/homebrew/)
	
Run it
------

1. Run the default Rake command

		rake 

2. Check the files in the `output` folder
